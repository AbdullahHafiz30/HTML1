HTML course

course description

لغة HTML هي طريقة لكتاب وهيكلة صفحات الويب، ولاتعتبر HTML لغة برمجية بالمعنى الفعلي بل هي لغة وصفيّة، أي أنها تستخدم لوصف الصفحة وتوزيع وترتيب محتواها من نصوص وروابط وصور ومقاطع فيديو وما إلى ذلك، وهي المكون الرئيسي لأي موقع ويب، تستخدم HTML مايسمى بالوسوم أو Tags وهي عبارة عن رموز لكل منها مهمة خاصة يقوم بها، فمنها ما يقوم بعرض العنوان ومنها ما يقوم بعرض النصوص ومنها ما يقوم بعرض الصور أو الروابط أو مقاطع الفيديو وهكذا، والمتصفح يفهم هذه الرموز أو الوسوم ويعرضها بالشكل المحدد والمطلوب

مواعيد المحاضرات اليومية

مستويات هذه الدورة
1st level: اساسيات HTML
2nd level: مفاهيم متقدمة في HTML

اهداف هذه الدورة

  1. التعرف على مفهوم HTML
  2. تعلم كل tag والغرض من استخدامه
  3. طرق اضافة النصوص؟، الفيديوهات، وغيرها من المحتويات المعتادة في صفحات الويب
